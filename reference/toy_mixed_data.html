Example Mixed-Type Data

toy_mixed_data.Rd

A small built-in mixed-type data frame for examples and tests.

Usage

toy_mixed_data

Format

A data frame with 60 rows and 6 variables:

sample_id

Sample identifier.

age

Continuous age-like feature.

score

Continuous score-like feature.

smoker

Binary factor.

stage

Ordered factor.

subtype

Nominal factor.
