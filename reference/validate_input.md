# Validate a Structured Input Table

Validate a Structured Input Table

## Usage

``` r
validate_input(data, id_column = NULL, min_rows = 5L)
```

## Arguments

- data:

  A data frame or matrix-like object.

- id_column:

  Optional identifier column name.

- min_rows:

  Minimum number of rows required.

## Value

A validated data frame.
