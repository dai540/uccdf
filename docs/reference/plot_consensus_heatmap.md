# Plot a Consensus Heatmap

Plot a Consensus Heatmap

## Usage

``` r
plot_consensus_heatmap(fit, k = NULL, ...)
```

## Arguments

- fit:

  A `uccdf_fit` object.

- k:

  Optional `K`.

- ...:

  Additional arguments passed to
  [`stats::heatmap()`](https://rdrr.io/r/stats/heatmap.html).

## Value

Invisibly returns `fit`.
