# Plot the Latent Embedding

Plot the Latent Embedding

## Usage

``` r
plot_embedding(fit, dims = c(1, 2), ...)
```

## Arguments

- fit:

  A `uccdf_fit` object.

- dims:

  Two dimensions to plot.

- ...:

  Additional arguments passed to
  [`graphics::plot()`](https://rdrr.io/r/graphics/plot.default.html).

## Value

Invisibly returns `fit`.
